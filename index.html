Chroma Linux Drivers

For Razer peripherals.

Features

An entirely open source driver, daemon and graphical frontend to manage your Razer peripherals on GNU/Linux systems.

Daemon

Background service that interfaces with your device.

DKMS Support

Recompiles the driver when the kernel is updated.

Fully Featured

Marcos on-the-fly, game mode, and profile support.

Chroma Configuration

Manage your peripherals and create key light profiles.

Tray Icon Included

Quickly adjust to suit your mood.

Scripts & More!

There's even a game of snake!

  Creating a package from source

1. Download the source code:

 git clone --depth=1 https://github.com/pez2001/razer_chroma_drivers.git
 cd razer_chroma_drivers

2. Install the packages needed to build the software.

 sudo apt-get install -y dpkg-dev libdbus-1-dev jq libsdl2-dev libsdl2-image-dev libfftw3-dev

3. Build the software and driver.

 make

4. Build the package.

   ./package_for_ubuntu.sh
   ./package_for_debian.sh

The command above will output something like: dpkg-name: info: moved 'tmp.3PnAtckx3o.deb' to '/tmp/razer-chroma-driver_1.0.0_amd64.deb'.

5. Install the newly created package:

 sudo dpkg -i /tmp/razer-chroma-driver*.deb

6. Reboot the system or use this command to activate the driver:

 sudo modprobe razerkbd

  Installing from source

This installs the components required to get the driver running. GUI applications will need to be manually installed.

 git clone --depth=1 https://github.com/pez2001/razer_chroma_drivers.git
 cd razer_blackwidow_chroma_driver
 ./install_driver_debian.sh

These steps will need to repeated when the repository is updated.

  Other Distributions

1. Install dependencies from your distribution:

  • libdbus-1-dev
  • jq
  • git
  • make
  • python3
  • python-appindicator

2. Obtain a copy of the source code:

 git clone --depth=1 https://github.com/pez2001/razer_chroma_drivers.git
 cd razer_chroma_drivers

3. Build and execute the installation script:

 sudo make -s all install

4. Once installed, reboot the system.


To use the latest changes from the repository, you will need to repeat the process:

 cd /path/to/razer_blackwidow_drivers/
 git pull

If at a later date you would like to remove the drivers, run:

 cd /path/to/razer_blackwidow_drivers/
 sudo make uninstall
